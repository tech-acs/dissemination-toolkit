# Rôles

Les rôles regroupent des permissions afin de pouvoir les affecter aux utilisateurs. Accessibles depuis le menu déroulant **Gérer** de la gestion sous **Contrôle d'accès → Rôles** (Super Admin uniquement).

## Création d'un rôle

1. Allez à la page Rôles depuis le menu d'administration de gestion
2. Saisissez un **Nom** pour le rôle dans le formulaire en haut de la page
3. Cliquez sur **Créer**
4. Le nouveau rôle apparaît dans le tableau des rôles ci-dessous

## Modification d'un rôle (matrice de permissions)

1. Cliquez sur **Modifier** à côté d'un rôle
2. La page affiche une matrice de permissions avec toutes les permissions disponibles regroupées par ressource :

| Ressource | Permissions |
|---|---|
| **Visualisations** | créer, modifier, publier et dépublier, supprimer |
| **Récits** | créer, modifier, publier et dépublier, supprimer |
| **Thèmes** | créer, modifier, supprimer |
| **Indicateurs** | créer, modifier, supprimer |
| **Dimensions** | créer, modifier, supprimer, gérer les valeurs |
| **Jeux de données** | créer, modifier, importer, publier et dépublier, supprimer |
| **Documents** | créer, modifier, supprimer, publier et dépublier |
| **Avis** | approuver |

3. Cochez les cases pour chaque permission que vous souhaitez accorder
4. Cliquez sur **Mettre à jour** pour enregistrer

## Suppression d'un rôle

1. Cliquez sur **Supprimer** à côté d'un rôle
2. Confirmez dans la boîte de dialogue
3. Le rôle **Super Admin** ne peut pas être supprimé

Chaque ligne de rôle indique également le nombre de permissions et d'utilisateurs affectés.
