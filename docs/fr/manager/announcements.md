# Annonces

Les annonces sont des messages diffusés envoyés à tous les utilisateurs d'un rôle sélectionné. Accessibles depuis le menu déroulant **Gérer** de la gestion en sélectionnant **Annonces**.

## Création d'une annonce

1. Allez à la page Annonces depuis le menu d'administration de gestion
2. Cliquez sur **Créer**
3. Remplissez les champs :
   - **Titre** — l'objet de l'annonce
   - **Destinataires** — sélectionnez un rôle ; tous les utilisateurs ayant ce rôle recevront la notification
   - **Message** — le corps de l'annonce
4. Cliquez sur **Envoyer**

Une fois envoyée, l'annonce apparaît dans la boîte de réception de notifications des destinataires sur le site public.
