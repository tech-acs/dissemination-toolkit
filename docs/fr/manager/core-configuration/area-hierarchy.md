# Hiérarchie des zones

Les hiérarchies de zones définissent les niveaux géographiques utilisés sur toute la plateforme, tels que Région, District, Comté ou Canton. Accessibles depuis le menu déroulant **Gérer** de la gestion sous **Configuration principale → Hiérarchie des zones** (Super Admin uniquement).

## Création d'un niveau de hiérarchie

1. Allez à la page Hiérarchie des zones depuis le menu d'administration de gestion
2. Cliquez sur **Créer**
3. Remplissez les champs :
   - **Nom** (multilingue) — ex. « Région », « District »
   - **Longueur de remplissage par zéros** — nombre de chiffres pour le remplissage par zéros des codes de zone
   - **Tolérance de simplification des formes** — valeur de tolérance pour simplifier les géométries des shapefiles lors de la diffusion des cartes
4. Cliquez sur **Enregistrer**

L'ordre des niveaux de hiérarchie détermine leur ordre d'affichage dans les filtres géographiques de toute la plateforme.

## Modification d'un niveau de hiérarchie

1. Cliquez sur **Modifier** à côté d'un niveau de hiérarchie
2. Mettez à jour les champs si nécessaire
3. Cliquez sur **Mettre à jour**

## Suppression d'un niveau de hiérarchie

1. Cliquez sur **Supprimer** à côté d'un niveau de hiérarchie
2. Confirmez dans la boîte de dialogue

La liste affiche des badges (Premier, Dernier) pour indiquer l'ordre des niveaux dans la hiérarchie.
