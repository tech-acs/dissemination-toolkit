# Indicateurs

Les indicateurs sont les points de données mesurables (ex. population, taux d'alphabétisation, chômage) qui apparaissent dans les jeux de données. Ce sont les éléments de base des visualisations — lors de la création d'un graphique, tableau, carte ou fiche synthétique, vous sélectionnez un ou plusieurs indicateurs d'un jeu de données à afficher. Accessibles depuis la navigation supérieure de gestion sous **Données → Indicateurs**.

## Création d'un indicateur

1. Allez à la page Indicateurs depuis la navigation supérieure de gestion
2. Cliquez sur **Créer**
3. Remplissez les champs :
    - **Nom** (multilingue)
    - **Description** (multilingue)
    - **Thèmes** — sélectionnez un ou plusieurs thèmes à associer
4. Cliquez sur **Enregistrer**

## Modification d'un indicateur

1. Cliquez sur **Modifier** à côté d'un indicateur dans la liste
2. Mettez à jour les champs si nécessaire
3. Cliquez sur **Mettre à jour**

## Suppression d'un indicateur

1. Cliquez sur **Supprimer** à côté d'un indicateur
2. Confirmez dans la boîte de dialogue
