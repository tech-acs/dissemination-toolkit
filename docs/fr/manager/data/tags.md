# Étiquettes

Les étiquettes sont des libellés qui peuvent être attachés aux visualisations, récits de données et documents pour le filtrage et la découverte. Accessibles depuis la navigation supérieure de gestion sous **Contenu → Étiquettes**.

Les étiquettes sont créées automatiquement lorsque les utilisateurs les saisissent dans le champ de saisie d'étiquettes sur les visualisations, récits de données et documents. Si une étiquette avec le même nom existe déjà, elle est réutilisée plutôt que dupliquée — le compteur d'utilisation est simplement incrémenté. Les étiquettes ne peuvent pas être créées directement depuis cette page, uniquement renommées.

## Consultation des étiquettes

La page d'index des étiquettes affiche un tableau avec :
- **Nom de l'étiquette**
- **Compteur d'utilisation** — nombre de visualisations, récits de données et tableaux de recensement utilisant cette étiquette

## Renommage d'une étiquette

1. Cliquez sur **Modifier** à côté d'une étiquette
2. Mettez à jour le **Nom** multilingue
3. Cliquez sur **Mettre à jour**

Le renommage s'applique à tout le contenu utilisant cette étiquette.
