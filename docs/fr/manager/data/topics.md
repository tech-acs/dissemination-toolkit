# Thèmes

Les thèmes sont utilisés pour catégoriser les indicateurs, visualisations et récits de données en groupes sémantiques. Lors de la création d'un indicateur, vous l'affectez à un ou plusieurs thèmes. Les visualisations héritent des thèmes de leurs indicateurs sélectionnés, tandis que les récits de données se voient affecter des thèmes directement. Sur le site public, les utilisateurs peuvent filtrer le contenu par thème sur les pages Explorateur de données, Visualisations et Récits de données. Accessibles depuis la navigation supérieure de gestion sous **Données → Thèmes**.

## Création d'un thème

1. Allez à la page Thèmes depuis la navigation supérieure de gestion
2. Cliquez sur **Créer**
3. Remplissez les champs :
    - **Nom** (multilingue)
    - **Description** (multilingue)
    - **Rang** — ordre d'affichage
4. Cliquez sur **Enregistrer**

## Modification d'un thème

1. Cliquez sur **Modifier** à côté d'un thème dans la liste
2. Mettez à jour les champs si nécessaire
3. Cliquez sur **Mettre à jour**

## Suppression d'un thème

1. Cliquez sur **Supprimer** à côté d'un thème
2. Confirmez dans la boîte de dialogue
