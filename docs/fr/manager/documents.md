# Documents

La section Documents vous permet de télécharger et gérer des tableaux de recensement publiés en fichiers Excel, avec des métadonnées Dublin Core complètes. Accessibles depuis la navigation supérieure de gestion sous **Contenu → Documents**.

## Création d'un document

1. Allez à la page Documents depuis la navigation supérieure de gestion
2. Cliquez sur **Créer**
3. Téléchargez le fichier (jusqu'à 100 Mo)
4. Sélectionnez le **Type** — Tableau de recensement, Document, Microdonnées ou Rapport
5. Remplissez les métadonnées :
   - **Titre** (multilingue)
   - **Description** (multilingue)
   - **Source des données** (multilingue)
   - **Producteur** (multilingue)
   - **Éditeur** (multilingue)
   - **Date de publication**
   - **Thèmes** — sélectionnez un ou plusieurs
   - **Étiquettes**
   - **Commentaire**
6. Cliquez sur **Enregistrer**

## Modification d'un document

1. Cliquez sur **Modifier** à côté d'un document
2. Mettez à jour le fichier, les métadonnées, ou les deux
3. Cliquez sur **Mettre à jour**

## Publication / dépublication

Basculez l'interrupteur de publication sur une ligne de document pour contrôler sa visibilité sur la page publique Documents. Nécessite la permission **Publier et dépublier**.

## Suppression d'un document

1. Cliquez sur **Supprimer** à côté d'un document
2. Confirmez dans la boîte de dialogue
