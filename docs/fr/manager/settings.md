# Paramètres (Image de marque)

La page Paramètres contrôle l'image de marque et les coordonnées de l'organisation affichées sur tout le site public. Accessible depuis le menu déroulant **Gérer** de la gestion en sélectionnant **Paramètres**.

## Modification des détails de l'organisation

1. Allez à la page Organisation depuis le menu d'administration de gestion
2. Mettez à jour les champs suivants :
   - **Nom** (multilingue)
   - **Slogan** (multilingue)
   - **Site web**
   - **Email**
   - **Logo** — téléchargez une image (format 16:9 recommandé)
   - **Introduction / Texte de présentation** (multilingue)
   - **Liens vers les réseaux sociaux** — X/Twitter, Facebook, Instagram, LinkedIn
   - **Adresse** (multilingue)
3. Cliquez sur **Mettre à jour**

Les modifications prennent effet immédiatement sur tout le site public, y compris le pied de page de la page d'accueil et l'image de marque de l'organisation.
