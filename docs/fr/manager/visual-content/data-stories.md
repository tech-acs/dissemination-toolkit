# Récits de données

Les récits de données sont des pages narratives riches qui combinent texte, images et visualisations intégrées pour raconter des histoires basées sur les données. Accessibles depuis la navigation supérieure de gestion sous **Contenu → Récits de données**.

## Création d'un récit

1. Allez à la page Récits de données depuis la navigation supérieure de gestion
2. Cliquez sur **Créer**
3. Remplissez les champs :

   | Champ | Description |
   |---|---|
   | **Titre** (multilingue) | Le titre du récit |
   | **Description** (multilingue) | Un résumé du récit |
   | **Thèmes** | Sélectionnez un ou plusieurs thèmes |
   | **Filtrable par géographie** | Basculer pour permettre aux utilisateurs de filtrer par zone |
   | **En vedette** | Basculer pour mettre en vedette sur la page d'accueil |
   | **Image en vedette** | Télécharger une image |
   | **Évaluable** | Basculer pour autoriser les évaluations et avis |
   | **Étiquettes** | Saisissez un nom d'étiquette et cliquez sur **Ajouter** |

4. Cliquez sur **Enregistrer**

## Conception du contenu du récit

Après avoir créé le récit, cliquez sur **Concevoir** pour ouvrir le concepteur de récits. L'en-tête de page affiche le titre du récit, la description et les boutons **Enregistrer** / **Réinitialiser** / **Fermer**.

### Section Ajouter un bloc

Cliquez sur n'importe quel bouton **Ajouter un bloc** pour ajouter du contenu au récit :

| Bloc | Description |
|---|---|
| **Texte** | Éditeur de texte riche avec barre d'outils de mise en forme. Prend en charge le format de bloc (Normal, Titre 1, Titre 2, Titre 3), gras, italique, souligné, liens, listes ordonnées et à puces, et nettoyage de la mise en forme. Chaque bloc de texte a un bouton de suppression **✕**. |
| **Image** | Télécharger une image avec légende facultative. Prend en charge l'alignement (gauche, droite, centre, pleine largeur). |
| **Visualisation** | Intégrer n'importe quel graphique, tableau, carte ou fiche synthétique publié. Sélectionnez dans une liste déroulante des visualisations disponibles. |
| **Conteneur deux colonnes** | Une disposition côte à côte. Chaque colonne peut contenir indépendamment du texte ou une visualisation intégrée. Les blocs internes ont leurs propres boutons de suppression **✕**. |

Les blocs peuvent être ajoutés, supprimés et réorganisés. Cliquez sur **Enregistrer** pour persister les modifications, **Réinitialiser** pour annuler, ou **Fermer** pour quitter sans enregistrer.

## Duplication d'un récit

1. Cliquez sur **Dupliquer** à côté d'un récit
2. Une copie du récit (y compris ses blocs de contenu) est créée, prête à être modifiée

## Publication / dépublication

Basculez l'interrupteur de publication sur une ligne de récit pour contrôler sa visibilité sur la page publique Récits de données. Nécessite la permission **Publier et dépublier**.

## Restriction / partage

Basculez l'interrupteur de restriction pour limiter un récit à son propriétaire uniquement, ou le rendre accessible à tous les utilisateurs. Nécessite la permission **Modifier**.

## Modification d'un récit

1. Cliquez sur **Modifier** à côté d'un récit
2. Mettez à jour les champs de métadonnées
3. Cliquez sur **Mettre à jour**

## Suppression d'un récit

1. Cliquez sur **Supprimer** à côté d'un récit
2. Confirmez dans la boîte de dialogue
