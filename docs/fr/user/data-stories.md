# Récits de données

Les récits de données sont des pages narratives riches qui combinent texte, images et visualisations intégrées pour raconter des histoires basées sur les données. Ils fonctionnent comme des rapports ou des articles de blog construits à partir de données en direct.

## Navigation

La page Récits de données affiche les récits publiés dans une disposition en cartes. Vous pouvez :

- **Rechercher** par mot-clé
- **Filtrer par thème** à l'aide du menu déroulant

Chaque carte affiche une image en vedette, un titre et une description.

## Lecture d'un récit

La page de détail d'un récit de données comprend :

- **En-tête** — thèmes, titre, description, étiquettes et date de publication
- **Filtre de zone** — si le récit est filtrable par géographie, vous pouvez sélectionner une zone pour filtrer les données
- **Blocs de contenu** — les récits sont composés de blocs flexibles :
  - **Texte** — contenu mis en forme riche
  - **Image** — images avec options d'alignement (gauche, droite, centre, pleine largeur) et légendes facultatives
  - **Visualisation** — graphiques, tableaux, cartes ou fiches synthétiques intégrés et entièrement interactifs
  - **Deux colonnes** — disposition côte à côte contenant n'importe quel mélange de blocs texte, image et visualisation
