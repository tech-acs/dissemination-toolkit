# Jeux de données

Les jeux de données sont les données structurées sous-jacentes — des tables de faits — qui alimentent toute la plateforme. Chaque jeu de données contient des indicateurs, des dimensions et des observations.

## Navigation

La page Jeux de données affiche les jeux de données publiés sous forme de grille de cartes. Vous pouvez :

- **Rechercher** par mot-clé (correspond au nom du jeu de données)
- **Filtrer par thème** à l'aide du menu déroulant

## Carte de jeu de données

Chaque carte affiche :

- **Nom** et **description** du jeu de données
- **Indicateurs inclus** — listés par nom
- **Désagrégé par** — les noms de dimensions disponibles pour le filtrage
- **Années d'observation** — la plage d'années couverte
- **Granularité** — le niveau géographique le plus fin disponible
- **Observations** — nombre total de lignes de données

## Actions

Chaque carte de jeu de données a deux boutons d'action :

| Action | Description |
|---|---|
| **Explorer** | Ouvre l'explorateur de données avec ce jeu de données présélectionné, prêt à être parcouru et filtré |
| **Télécharger** | Télécharge le jeu de données sous forme de fichier structuré (CSV/Excel) |
