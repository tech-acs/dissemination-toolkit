# Documents

La section Documents contient des tableaux de recensement publiés en fichiers Excel, organisés par année de recensement et par série. Ces tableaux maintiennent une conception cohérente au fil du temps pour favoriser la comparabilité des résultats du recensement.

## Navigation

La page Documents propose plusieurs moyens pour trouver les tableaux dont vous avez besoin :

### Filtres (barre latérale gauche)

| Filtre | Description |
|---|---|
| **Type** | Filtrer par type de document : Tableau de recensement, Document, Microdonnées ou Rapport |
| **De l'année / À l'année** | Filtrer par plage d'années de recensement |
| **Thème** | Filtrer par thème |
| **Étiquettes** | Sélectionner des étiquettes pour affiner les résultats |

### Recherche

Utilisez la barre de recherche pour trouver des tableaux par mot-clé dans le titre.

### Tri

Les résultats peuvent être triés par Pertinence, Popularité, Récent, Ancien, Titre (A-Z) ou Titre (Z-A).

## Liste des documents

Chaque document dans la liste des résultats affiche :

- Icône de type et titre (lien vers la page de détail)
- Source des données et éditeur
- Dates de création et de dernière modification
- Nombre de vues et de téléchargements
- Mois et année de publication
- Lien **Télécharger** indiquant le type et la taille du fichier

## Détail d'un document

Cliquez sur un document pour voir sa page de détail, qui comprend la description complète, la date de publication, la date de dernière modification, l'auteur de la modification, le producteur, l'éditeur, la source des données, le commentaire, les badges de thème et d'étiquette, et un bouton de téléchargement.
