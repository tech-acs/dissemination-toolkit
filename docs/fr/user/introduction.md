# Introduction

Le Dissemination Toolkit est une plateforme web pour la publication et l'exploration de données de recensement et d'enquêtes. Il est développé et maintenu par la **Commission économique des Nations Unies pour l'Afrique (CEA)**.

## Page d'accueil

Lors de votre première visite sur le site, vous arrivez sur la **page d'accueil**. Elle comprend :

- Une **section hero** décrivant la mission de la plateforme : vous aider à trouver, explorer et visualiser des données
- **Statistiques clés** — le nombre total de jeux de données, d'indicateurs, de visualisations et de récits de données publiés
- Un **appel à l'action** pour essayer l'explorateur de données
- **Récits de données en vedette** — pages narratives sélectionnées qui présentent des récits basés sur les données à l'aide de graphiques, tableaux, cartes et plus encore

## Sections principales

La barre de navigation supérieure donne accès aux sections principales de la plateforme :

| Section | Description |
|---|---|
| **Explorateur de données** | Interface par pointage et clic pour parcourir, filtrer, pivoter et télécharger des données de recensement |
| **Visualisations** | Parcourir les graphiques, tableaux, cartes et fiches synthétiques publiés |
| **Récits de données** | Pages narratives riches combinant texte, images et visualisations intégrées |
| **Documents** | Rechercher et télécharger des tableaux de recensement publiés en fichiers Excel |
| **Jeux de données** | Parcourir les jeux de données publiés et leurs métadonnées |

## Connexion

Le bouton **Connexion** dans le coin supérieur droit est le point d'entrée pour les gestionnaires et administrateurs. Après connexion, les utilisateurs autorisés peuvent accéder au tableau de bord de gestion pour créer et gérer le contenu, configurer la plateforme et contrôler l'accès des utilisateurs.

## Pour commencer

Si vous souhaitez explorer les données, commencez par l'**[Explorateur de données](/fr/user/data-explorer)**. Pour voir les visualisations ou les récits de données publiés, parcourez les sections respectives dans la barre de navigation.
