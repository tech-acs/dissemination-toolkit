# Visualisations

Les visualisations sont des graphiques, tableaux, cartes et fiches synthétiques publiés, créés à partir des données de la plateforme.

## Types

| Type | Description |
|---|---|
| **Graphique** | Graphiques interactifs — barres, lignes, nuages de points, secteurs, histogrammes, boîtes à moustaches, aires |
| **Tableau** | Tableaux de données triables |
| **Carte** | Cartes choroplèthes montrant la distribution géographique des données |
| **Fiche synthétique** | Un seul chiffre ou indicateur mis en évidence |

## Navigation

La page Visualisations affiche les visualisations publiées sous forme de grille de cartes. Vous pouvez :

- **Rechercher** par mot-clé (correspond au titre et aux étiquettes)
- **Filtrer par thème** à l'aide du menu déroulant

## Consultation d'une visualisation

Cliquez sur une visualisation pour ouvrir sa page de détail, qui comprend :

- Titre, description, étiquettes et date de publication
- La **visualisation interactive** — graphique, tableau, carte ou fiche synthétique
- Bouton **Intégrer** — génère le code d'intégration pour l'afficher sur des sites externes
- Bouton **Partager** — partager par email, Telegram, WhatsApp, Twitter/X, LinkedIn, Facebook, Microsoft Teams ou SMS
- **Évaluations et avis** — si la visualisation est évaluable, vous pouvez voir la note moyenne et lire les avis approuvés
- **Filtre de zone** — si la visualisation est filtrable par géographie, vous pouvez sélectionner une zone et l'appliquer pour filtrer les données affichées

## Étiquettes

Les étiquettes sont des libellés attachés aux visualisations pour faciliter leur découverte. Elles apparaissent sous forme de badges bleus sous la description sur la page de détail d'une visualisation.

Lorsque vous effectuez une recherche par mot-clé sur la page Visualisations, la recherche correspond à la fois aux titres des visualisations **et** à leurs étiquettes — ainsi, taper un nom d'étiquette comme « population » fera ressortir toutes les visualisations étiquetées avec celle-ci.
