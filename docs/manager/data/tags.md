# Tags

Tags are labels that can be attached to visualisations, data stories, and documents for filtering and discovery. Accessible at `manage/tag` (Super Admin only).

Tags are created automatically when users add tags to content via the tag input component. They cannot be created directly, only renamed.

## Viewing tags

The tags index page displays a table with:
- **Tag name**
- **Usage count** — how many visualisations, data stories, and census tables use this tag

## Renaming a tag

1. Click **Edit** next to a tag
2. Update the multi-language **Name**
3. Click **Update**

The rename applies to all content using that tag.
