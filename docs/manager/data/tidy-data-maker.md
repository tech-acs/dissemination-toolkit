# Tidy Data Maker

The Tidy Data Maker is a helper tool that prepares messy or non-tidy data into a format suitable for import into datasets. Accessible at `manage/tidy`.

## Using the Tidy Data Maker

1. Navigate to `manage/tidy`
2. The Livewire component provides an interface to load, transform, and export your data
3. Follow the on-screen instructions to pivot, reshape, and clean your data
4. Export the resulting tidy dataset for import via the **Datasets** section (`manage/dataset`)
