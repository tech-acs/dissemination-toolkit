# Documents

The Documents section lets you upload and manage census tables published as Excel files, with full Dublin Core metadata. Accessible at `manage/document`.

## Creating a document

1. Navigate to `manage/document`
2. Click **Create**
3. Upload the file (up to 100MB)
4. Select the **Type** — Census Table, Document, Microdata, or Report
5. Fill in the metadata:
   - **Title** (multi-language)
   - **Description** (multi-language)
   - **Data source** (multi-language)
   - **Producer** (multi-language)
   - **Publisher** (multi-language)
   - **Published date**
   - **Topics** — select one or more
   - **Tags**
   - **Comment**
6. Click **Save**

## Editing a document

1. Click **Edit** next to a document
2. Update the file, metadata, or both
3. Click **Update**

## Publishing / unpublishing

Toggle the publish switch on a document row to control its visibility on the public Documents page. Requires the `publish-and-unpublish:document` permission.

## Deleting a document

1. Click **Delete** next to a document
2. Confirm in the dialog
