# Data Stories

Data Stories are narrative-rich pages that combine text, images, and embedded visualizations to tell data-driven stories. They function like reports or blog posts built from live data.

## Browsing

The Data Stories page displays published stories in a card layout. You can:

- **Search** by keyword
- **Filter by topic** using the dropdown

Each card shows a featured image, title, and description.

## Reading a story

A Data Story detail page includes:

- **Header** — topics, title, description, tags, and publication date
- **Area filter** — if the story is filterable by geography, you can select an area to filter the data
- **Content blocks** — stories are composed of flexible blocks:
  - **Text** — rich formatted content
  - **Image** — images with alignment options (left, right, center, full-width) and optional captions
  - **Visualization** — embedded charts, tables, maps, or scorecards that are fully interactive
  - **Two-column** — side-by-side layout containing any mix of text, image, and visualization blocks
