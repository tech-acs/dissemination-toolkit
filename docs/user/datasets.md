# Datasets

Datasets are the underlying structured data — fact tables — that power the entire platform. Each dataset contains indicators, dimensions, and observations.

## Browsing

The Datasets page displays published datasets in a card grid. You can:

- **Search** by keyword (matches dataset name)
- **Filter by topic** using the dropdown

## Dataset card

Each card shows:

- **Dataset name** and **description**
- **Indicators included** — listed by name
- **Disaggregated by** — the dimension names available for filtering
- **Observation years** — the range of years covered
- **Granularity** — the highest geographic level available
- **Observations** — total count of data rows

## Actions

Each dataset card has two action buttons:

| Action | Description |
|---|---|
| **Explore** | Opens the Data Explorer with this dataset pre-selected, ready for you to browse and filter |
| **Download** | Downloads the dataset as a structured file (CSV/Excel) |
