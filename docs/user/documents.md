# Documents

The Documents section contains census tables published as Excel files, organised by census year and series. These tables maintain a consistent design over time to foster comparability of census outputs.

## Browsing

The Documents page provides several ways to find the tables you need:

### Filters (left sidebar)

| Filter | Description |
|---|---|
| **Type** | Filter by document type: Census Table, Document, Microdata, or Report |
| **From year / To year** | Filter by census year range |
| **Topic** | Filter by topic |
| **Tags** | Select tags to narrow results |

### Search

Use the search bar to find tables by title keyword.

### Sort

Results can be sorted by relevance, popularity, date (newest/oldest), or title (A-Z/Z-A).

## Document list

Each document in the results list shows:

- Type icon and title (linked to the detail page)
- Data source and publisher
- Creation and last modified dates
- View and download counts
- Published month and year
- **Download** link showing file type and size

## Document detail

Click a document to view its detail page, which includes the full description, metadata (producer, publisher, data source), clickable topic and tag links, and a download button.
