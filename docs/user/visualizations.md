# Visualizations

Visualizations are published charts, tables, maps, and scorecards created from the platform's data.

## Types

| Type | Description |
|---|---|
| **Chart** | Interactive charts — bar, line, scatter, pie, histogram, box, area |
| **Table** | Sortable data tables |
| **Map** | Choropleth maps showing geographic data distribution |
| **Scorecard** | A single highlighted number or indicator |

## Browsing

The Visualizations page displays published visualizations as a card grid. You can:

- **Search** by keyword (matches title and tags)
- **Filter by topic** using the dropdown

## Viewing a visualization

Click a visualization to open its detail page, which includes:

- Title, description, tags, and publication date
- The **interactive visualization** — chart, table, map, or scorecard
- **Embed** button — generates embed code for embedding on external sites
- **Share** button — share via Email, Telegram, WhatsApp, Twitter/X, LinkedIn, Facebook, Microsoft Teams, or SMS
- **Ratings & reviews** — if the visualization is reviewable, you can see the average rating and read approved reviews
- **Area filter** — if the visualization is filterable by geography, you can select an area and apply it to filter the displayed data
